Class: InteractiveADXShell

InteractiveADXShell(dir, optionsopt)

new InteractiveADXShell(dir, optionsopt)

Create an interactive spawn process with the ADXShell. Manage the ADXShell process in interactive mode. It allow a single instance creation of the ADXShell and a bi-directional communication using the stdio of the spawn process
